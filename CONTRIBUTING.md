# Contributing to Terraflow

Thank you for your interest in contributing to **Terraflow**!

We welcome contributions that improve Terraflow, fix bugs, or add features.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your work.
3. Include tests if applicable.
4. [Open a pull request](https://help.github.com/articles/creating-a-pull-request-from-a-fork/).

## Contribution Guidelines

- Follow standard Go conventions.
- Write clear commit messages. Follow the [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/) convention.
- Mark as Draft to get an early review, even if it's not ready.
